# 🌱 SustentAgro

O **SustentAgro** é uma aplicação web desenvolvida com o objetivo de conectar empresas do setor agrícola a organizações não governamentais (ONGs), facilitando a doação de alimentos que não atendem aos padrões comerciais ou que representam excedentes de produção.

A plataforma busca reduzir o desperdício de alimentos e, ao mesmo tempo, contribuir para ações sociais, promovendo um impacto positivo tanto no meio ambiente quanto na sociedade.

---

## 🎯 Objetivo

O projeto tem como finalidade:

- Reduzir o desperdício de alimentos no setor agrícola  
- Incentivar a doação de excedentes de produção  
- Conectar empresas rurais a ONGs e iniciativas sociais  
- Promover práticas sustentáveis no agronegócio  

---

## 🧩 Funcionalidades

- 📄 Apresentação de ONGs disponíveis para doação  
- 🌾 Divulgação de iniciativas sustentáveis  
- 🔍 Visualização de informações sobre as organizações  
- 💡 Interface simples e acessível para navegação
